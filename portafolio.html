Compilador Skibidi K+

Skibidi K+ es un compilador completo desarrollado en C# Windows Forms para un lenguaje de programación diseñado desde cero. Implementa todas las fases clásicas de compilación: análisis léxico (autómatas finitos), análisis sintáctico (GLC), análisis semántico (validación de tipos), generación de código intermedio (lenguaje ensamblador), optimización y generación de código objeto (ejecución de programas).

Diseño de cableado estructurado para institución educativa

Basada en estándares de cableado estructurado y segmentación jerárquica mediante MDF (Main Distribution Frame) e IDF (Intermediate Distribution Frame) la propuesta contempla una topología centralizada con backbone de fibra óptica para interconectar 100 nodos, garantizando cumplimiento de la norma ANSI/TIA-568 (no más de 90 metros de enlace). Esta red incluye:

  • Switches administrables
  • Router principal
  • Servidor para control y almacenamiento de aplicaciones
  • Sistema de videovigilancia (NVR y cámaras IP con PoE)
  • Puntos de acceso inalámbricos estratégicamente distribuidos
  • Selección de materiales (rack, patch panels, transceivers, jumpers, organizadores)

Monitor inteligente de calidad del aire (IoT)

Desarrollo de un sistema IoT para monitoreo ambiental en tiempo real utilizando ESP32, sensores DHT11 y MQ-135, almacenamiento en Firebase Realtime Database y visualización mediante aplicación móvil en .NET MAUI. Incluye transmisión WiFi, consumo de API REST, deserialización JSON y cálculo estimado de AQI.

Sistema de Gestión HoneyLens

Sistema de gestión para estudio fotográfico desarrollado en C# con SQL Server para la gestión integral de un estudio fotográfico, enfocado en centralizar procesos operativos, comerciales y financieros bajo una sola plataforma.

  • Registro y autenticación de clientes, fotógrafos y administradores, con permisos diferenciados y paneles personalizados según el tipo de usuario.
  • Programación de sesiones, asociación de fotos a cada cliente y sistema de reseñas y calificaciones.
  • Creación y gestión de paquetes y promociones, registro de ventas e historial.
  • Monitoreo de ingresos, gastos, sueldos e inventario, con generación de reportes mensuales.

Sistema de Gestión de Eventos

Aplicación web desarrollada en ASP.NET Core que integra la librería FullCalendar para la gestión y visualización interactiva de eventos. Permite administrar fechas y horarios con distintas vistas (mensual, semanal y lista), combinando lógica de servidor con renderizado dinámico en el cliente.
